dotViewer Icon Design Reference

Apple-style macOS icon concepts — 2025 design trends

Concept A: Code Inspector

Recommended
Single metaphor: A magnifying glass inspecting code lines on an indigo background. The highlighted line inside the lens draws focus. Faint code lines in the background add context without clutter. This is the most "Apple" of all concepts — clean, recognizable at any size, and tells the story of "viewing code" instantly.

Concept B: The Dot Lens

Alternative
.
Playful and bold: A magnifying glass with a prominent "dot" (period character) at its center. This literally spells out "dotViewer" — a lens viewing a dot. The simplicity is striking but may not immediately communicate "code viewer" to new users.

Concept C: Code Brackets

Minimal
{
}
Ultra-minimal: Curly braces framing a glowing dot on a deep indigo background. This is the most typography-driven approach. Inspired by how Terminal and code editors use simple symbols. Works beautifully at all sizes due to its simplicity.

Concept D: Document Eye

Alternative
Metaphor blend: A document page with an eye symbol embedded at the bottom, suggesting "watching" or "viewing" the document contents. The eye adds personality while the document shape grounds the concept in file viewing.

Apple Icon Design Principles Applied

Single Metaphor

Each concept uses ONE clear visual idea. No competing symbols. Your current icon has eye + magnifier + brackets + code bars — that's 4 metaphors. Pick one.

Squircle Shape

Icon Composer automatically applies the macOS squircle mask. Design your content within this boundary. Leave ~12% padding from edges.

Limited Palette

Use 1-2 primary colors maximum. The indigo-to-violet gradient is distinctive and associates well with developer tools (see Xcode, Instruments).

Scale Gracefully

The icon must read clearly from 1024px down to 16px. Test at dock size (48-64px). If details vanish, simplify.

Subtle Depth

Use gentle inner highlights (top-left) and shadows (bottom-right) to suggest materiality. Avoid heavy 3D effects — Apple uses "refined realism."

Liquid Glass Ready

macOS 26 Tahoe introduces Liquid Glass. Design with clear foreground/background separation. Icon Composer lets you create layers for this.
